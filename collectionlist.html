Collection List

A Sample list object supports the following methods:
Add(obj) Adds an object to the end of the list.
Clear() Clears all the objects in the list.
Count() Gets the number of objects in the list.
IndexOf(obj) Returns the zero-based index of the first occurrance of the object in the list.
InsertAt(obj, index) Inserts an object to a specific index.
Item(index) Returns the object at an specific index.
Remove(obj) Removes the first occurrance of the object from the list.
RemoveAt(index) Removes the object from the specific index.
